Petripy
=======

compile annotated Petri nets into Arduino code

Python 2.7, tested under Windows 7 but should work anywhere. Refer to the documentation file for details. 
Find here http://www.walteraprile.com/wordpress/?p=46 some remarks on tools to manipulate Petri nets.
